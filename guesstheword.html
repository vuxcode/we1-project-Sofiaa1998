How to play the game

You start the game by clicking the Guess The Word alternative up in the menu bar. The game starts right away. A random word is generated which is hidden by a underline per letter. If you guess a letter wrong for a fifth time, you will lose the game. The number of how many times you can guess wrong before you lose the game is shown on the screen. Try a new word simply by pressing the New Game button.
